# Financial Data Analytics Terminal

## Português

### Descrição
Este projeto é um terminal de monitoramento financeiro desenvolvido em Python. O sistema realiza a coleta automatizada de cotações de moedas (USD, EUR, GBP, JPY) via API, processa os dados utilizando inteligência artificial para análise de sentimento de mercado e armazena o histórico em arquivos Excel para persistência de dados. A interface é construída em Streamlit, oferecendo visualizações gráficas interativas e ferramentas de conversão monetária.

### Tecnologias Utilizadas
- Linguagem: Python 3.13
- Interface: Streamlit
- Processamento de Dados: Pandas
- Armazenamento: Openpyxl (Excel)
- Inteligência Artificial: TextBlob (NLP)
- Gráficos: Plotly

### Aviso de Segurança e Isenção de Responsabilidade
Os dados apresentados são obtidos de fontes públicas e podem sofrer atrasos. O autor não se responsabiliza por decisões financeiras tomadas com base nas informações deste painel. Investimentos envolvem risco e este terminal não deve ser utilizado como única ferramenta de análise ou aconselhamento financeiro profissional.

## English

### Description
This project is a financial monitoring terminal developed in Python. The system performs automated data collection of currency exchange rates (USD, EUR, GBP, JPY) via API, processes the data using artificial intelligence for market sentiment analysis, and stores the history in Excel files for data persistence. The interface is built with Streamlit, providing interactive graphical visualizations and currency conversion tools.

### Tech Stack
- Language: Python 3.13
- Interface: Streamlit
- Data Processing: Pandas
- Storage: Openpyxl (Excel)
- Artificial Intelligence: TextBlob (NLP)
- Graphics: Plotly
